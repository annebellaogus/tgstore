# 🛒 Telegram Session Store Bot

A professional Telegram bot for selling Telegram sessions/aged accounts with admin panel, payment system, and auto-delivery.

## ✨ Features

- 🛒 **Product Store** with categories
- 💰 **Multiple Payment Methods** (Crypto, UPI)
- 📦 **Stock Management** system
- 👤 **User Management** with ban/unban
- 📊 **Admin Panel** with statistics
- 📢 **Broadcast** messages
- 🔒 **Force Join** channel/group
- 📋 **Order Management** with delivery system
- 🔄 **Replacement Policy** support
